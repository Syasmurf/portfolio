# Syamimi Kamaruddin - Professional Portfolio

A modern, professional machine learning-themed portfolio website showcasing skills, projects, and experience.

## 🎨 Features

### Professional Design
- **ML-Themed Color Scheme**: Neon blue (#00f5ff), purple (#a855f7), and pink (#ec4899)
- **Dark Mode Design**: Professional dark background with glassmorphism effects
- **Animated Backgrounds**: Gradient animations and floating particles
- **Smooth Transitions**: Cubic-bezier animations throughout

### Advanced Features
- ✨ **Glassmorphism Cards**: Modern frosted glass effect on all cards
- 🎭 **3D Hover Effects**: Interactive card transformations on hover
- 📱 **Fully Responsive**: Optimized for all devices (mobile, tablet, desktop)
- 🎯 **Smooth Scrolling**: Enhanced navigation with offset for fixed navbar
- 🔍 **Active Section Highlighting**: Auto-highlights current section in navigation
- ⬆️ **Scroll-to-Top Button**: Appears when scrolling down
- ⌨️ **Keyboard Navigation**: Esc closes menu, Ctrl+Arrow navigates sections
- 🎬 **Scroll Animations**: Elements fade in as you scroll
- 🎨 **Custom Scrollbar**: Styled to match the theme

### Sections

4. **Experience** - Professional work experience
5. **Skills** - Technical skills organized by category
